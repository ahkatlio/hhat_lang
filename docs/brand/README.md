# H-hat Brand Assets

This folder contains visual identity guidelines and brand assets for the H-hat quantum programming language project.

  - Logo specifications and usage
  - Color palette (light and dark themes)

  - Syntax highlighting colors
  - Accessibility guidelines
  - Brand application examples

## Usage

When using H-hat branding:

1. **Read the guidelines**: See [identity.md](identity.md) for complete specifications
2. **Use official assets**: Don't recreate or modify the logo
3. **Maintain colors**: Use the defined palette for consistency
4. **Give attribution**: Logo is CC BY-SA 4.0, requires credit
